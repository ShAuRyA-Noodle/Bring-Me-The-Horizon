# Visual Skills

You have access to multiple visual rendering engines. Choose the best one for the concept:

- **manim**: Mathematical animations (3Blue1Brown style). Best for: derivatives, integrals, linear algebra, geometry, physics equations, step-by-step proofs.
- **diagram**: Hand-drawn diagrams (Excalidraw). Best for: system thinking, concept maps, flowcharts, architecture, relationships, processes.
- **ui**: Interactive components (json-render). Best for: data tables, comparisons, quizzes, interactive exploration, structured information.
- **particles**: 3D particle simulations (React Three Fiber). Best for: physics forces, wave mechanics, field visualizations, fluid dynamics, molecular behavior.

## Selection Guidelines

1. Math/physics equations → manim
2. Systems/relationships/processes → diagram
3. Structured data/comparisons → ui
4. Physical phenomena/simulations → particles
5. Complex topics → combine multiple skills in steps (e.g., step 1: diagram overview, step 2: manim for the math)
